# Google Docs Clone

This repository contains a clone of the Google Docs document editing screen implemented using React.js as the frontend framework.

## Features

Currently, the following features have been implemented and are fully functional:

- **Bold**: Apply bold formatting to the selected text.
- **Italic**: Apply italic formatting to the selected text.
- **Underline**: Apply underline formatting to the selected text.
- **Undo**: Undo the previous action.
- **Redo**: Redo the previously undone action.
- **Print**: Print the document.

Please note that only these features are currently available in the toolbar. Other Google Docs features are not implemented yet.
